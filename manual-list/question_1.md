### Test #1: Automatic Behavior

1. Does any content **update automatically**? Pay special attention to components such as carousels and message feeds.
   
   - **YES**: Can the user **pause, stop, or hide** these updates, or else **control their frequency**? If not, raise an issue (2.2.2, 2.2.2).
2. Does any content **move, blink, or scroll**?
   
   - **YES**: Does the moving, blinking, or scrolling last for **more than 5 seconds**?
     
     - **YES**: Is there a mechanism for the user to **pause, stop, or hide** the moving, blinking, or scrolling content? If not, raise an issue (2.2.2).
3. Does any content **flash** more than 3 times per second?
   
   - **YES**: Does the flashing content exceed 25% of any 10-degree visual field on the screen at a typical viewing distance? If so, raise an issue (2.3.1).
