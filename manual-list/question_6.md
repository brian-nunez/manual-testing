### Test #6: Sensory Characteristics

1. Does the page use **color** to convey information, indicate an action, prompt a response, or distinguish a visual element?
   
   - **YES**: In each instance, is there a different **visual cue**[1](#footnote-sensory-characteristics-1) (that does *not* rely on color) that conveys the **same information**? If not, raise an issue (1.4.1, 1.4.1).
2. Does the page provide **instructions** for operating or understanding the content that require the ability to perceive **visual characteristics**, such as shape, color, size, visual location, or orientation, in order to be followed? An example would be "Select all the images of stop signs".
   
   - **YES**: In each instance, are **equivalent instructions** conveyed in a manner that does not require the ability to perceive visual characteristics in order to be followed? If not, raise an issue (1.1.1, 1.3.3).
3. Does the page provide **instructions** for operating or understanding the content that require the ability to perceive **auditory characteristics** in order to be followed? An example would be "Click the 'Start' button when you hear the sound of the bell".
   
   - **YES**: In each instance, are **equivalent instructions** conveyed in a manner that does not require the ability to perceive auditory characteristics in order to be followed? If not, raise an issue (1.3.3).

[\[1\]](#footnote-trigger-sensory-characteristics-1): Note that text content is an acceptable form of visual cue.
